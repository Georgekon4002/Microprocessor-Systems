# Συστήματα Μικροεπεξεργαστών (6ο εξάμηνο ΗΜΜΥ ΕΜΠ)

**Κωνσταντόπουλος Γιώργος (03122104)**

**Ηλίας Πουλόπουλος (03122160)**

Το παρόν αποθετήριο περιέχει όσα προγράμματα **assembly** ή και embedded **C** γράφτηκαν για τον **8085** ή για τον **ATMega328PB** για τις ασκήσεις του ακαδημαϊκού έτους 2024-2025. Περιέχει επιπλέον τις εκφωνήσεις των εν λόγω ασκήσεων, αλλά και κάποιες οδηγίες που δόθηκαν σχετικά με τις εντολές, αλλά και τα προγράμματα που έπρεπε να εγκαταστήσουμε **(MPLAB X IDE ή Microchip Studio)**.
